HTML Tags

Emphasis
<em></em> tags are used to emphasize the text in Italic.
Quote
<q></q> tags are used to put Quote with quotation marks.
Highlighting text
<mark></mark> tags are used to highlight the text like using highlighter on paper.
Subscript text
<sub></sub> tags are used to set the text slightly below the regular text.
Superscript text
<sup></sup> tags are used to set the text slightly above the regular text.
